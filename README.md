# クエスト掲示板 開発ガイド

このプロジェクトは、モノレポ構成（frontend / backend）で構築されたクエスト投稿・参加アプリです。

---

## 🛠 環境構築手順

### 1. リポジトリをクローン

```bash
git clone https://github.com/your-org/quest-board-app.git
cd quest-board-app
```

### 2. 依存パッケージをインストール

```bash
pnpm install
```

---

## 🚀 アプリの動かし方

### フロントエンド

```bash
pnpm dev
```

### バックエンド

```bash
pnpm --filter backend dev
```
