# 汎用 AI 実行プロンプト

このファイルは、Claude / Codex / Cursor などのエージェントに渡すための短い実行テンプレートである。リポジトリの正本ルール自体は `AGENTS.md`、`docs/architecture.md`、`docs/ai-execution.md` に置く。

## 目的

- 別の AI エージェントにも同じ参照順序を渡す
- 実行前提を短く共有する
- ルール本文の重複を避ける

## AI が読むタイミング

- 外部エージェントにタスクを渡す直前
- 追加のコンテキストを与えたいとき

## テンプレート

```md
あなたはこのリポジトリで作業する AI エージェントです。

作業開始前に以下を順に確認してください。

1. README.md
2. AGENTS.md
3. docs/architecture.md
4. docs/ai-execution.md
5. 必要に応じて関連 prompt
6. 関連コードと関連テスト

守るべき原則:

- 既存実装と既存差分を先に確認する
- 前提を決め打ちしない
- issue 対応では必要に応じて worktree で作業を分離する
- 最小変更で実装する
- unrelated な修正を混ぜない
- 実装後は可能な範囲でテスト、型チェック、lint、ビルドを行う
- 実行できなかった検証は理由を報告する
- 判断不能な場合だけ質問し、それまでは既存コードから仮説を立てて進める

今回の依頼:

<ここにユーザー依頼を貼る>
```

## 含める内容

- 参照順序
- 最低限の行動原則
- タスク差し込み用の雛形

## 含めない内容

- アーキテクチャの詳細説明
- 実行フェーズの完全版
- リポジトリ固有の実装詳細の再掲
